# 1. 환경
### 1) 프론트엔드
- vue - 2.6.1
- npm
- node - v16.15.0

### 2) 백엔드
- springboot - 2.7.0
- java 11
- 의존성 - lombok, mustache

### 1) springboot
```
루트 디렉토리 임포트 후
```

```
frontend 폴더로 이동

npm i

npm run serve
```
